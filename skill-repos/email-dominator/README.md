## ğŸ”§ Troubleshooting

### Agent isn't using the skill

1. **Verify file location**: Make sure `SKILL.md` is in your agent's `skills/` folder
2. **Check file format**: The file should be plain text/markdown, not a Word doc
3. **Restart agent**: Some agents need a restart to pick up new skills
4. **Check logs**: Look for any error messages when the agent starts

### Skill triggers at wrong times

The skill activates when you mention: email, cold outreach, follow-up, customer support, sales emails, email templates, email marketing, newsletters, email responses

If it's triggering incorrectly, try being more specific with your requests.

### Emails sound too generic

Try providing more context:
- Who is the recipient? (role, industry, company size)
- What's your relationship? (cold contact, existing customer, etc.)
- What's your goal? (meeting, sale, information, etc.)
- What's your value proposition?

The more context you provide, the more personalized the emails become.
